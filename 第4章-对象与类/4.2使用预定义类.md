# 4.2.1对象与对象变量
新实列一般由构造器(constructor,也称为构造函数)构造,比如java标准库中有一个Date类,用于描述时间点,比如说   
Date birthday=**new Date()**;//构造器前加new操作符
上面的birthday是对象变量,并且引用了一个对象,但下面  
Date deadline;//没引用任何对象  
所以不能在dealline对象变量上使用任何方法.想要使用类的非静态方法必须要初始化,一般有一下三种选择  
1.  deadline=new Date();//引用一个新的对象  
2.  deadline=birthday;//引用一个已存在对象  
3.  deadline=null;//设置为null,表示没有引用任何对象  
注意Date deadline;这条语句不会将deadline设置为null,如果你之后没初始化它使用会报错  
```` java
import java.util.Date;
public class DateTest {
    public static void main(String[] args) {
        Date birthday;
        System.out.print(birthday==null);//error,会提示你birthday没初始化
    }
}
````

# 4.2.2java类库中的LocalDate类 && 4.2.3更改器方法与访问器方法
Date类表示时间点,还有一个是LocalDate类,用来表示大家熟悉的日历表示法。你可以使用构造器，也可以使用静态**工厂方法**,如  
LocalDate.now()会创造一个新对象，表示构造这个对象日期(工厂方法名称不需要是类名，含义更加清晰,比如now就比LocalDate在这里意义清晰,当然LocalDate**没有构造器**)  

```` java
import java.time.LocalDate;

public class DateTest {
    public static void main(String[] args) {
        LocalDate now_time = LocalDate.now();

        System.out.println(now_time.toString());// 打印格式为年-月-日(如2021-03-12)
        System.out.println("今年是" + now_time.getYear());
        // 方法getyear返回年数(int类型),类似的有getMonthValue,getDayOfMonth方法

        LocalDate aThousandDaysLater = now_time.plusDays(1000L);
        // plusdays方法没有修改对象，是访问器方法

        System.out.println("1000天后为" + aThousandDaysLater.toString());
        System.out.println("但是现在仍为" + now_time);// now_time没有被plusDays修改
    }
}
````
只访问对象而不修改对象的方法称为访问器方法(accessor method),修改对象的称为更改器方法(mutator method),二者在语法上没有多大区别(c++中可以用区分),虽然java可以在方法前加final，但主要是当前类被子类继承后,不允许**子类对该final方法重写**


```` java
import java.time.*;

/**
 * @version 1.5 2015-05-08
 * @author Cay Horstmann
 */
public class CalendarTest {
   public static void main(String[] args) {
      LocalDate date = LocalDate.now();
      int month = date.getMonthValue();
      int today = date.getDayOfMonth();

      date = date.minusDays(today - 1); // set to start of month，日期减去多少天后再返回副本，这里将date1设置成每月第一号
      DayOfWeek weekday = date.getDayOfWeek();// 返回一个枚举类型DayOfWeek ,代表date所表达日期是星期几
      int value = weekday.getValue(); // 1 = Monday, . . . , 7 = Sunday

      System.out.println("Mon Tue Wed Thu Fri Sat Sun");
      for (int i = 1; i < value; i++)// 比如说第一天是星期三，那么这之前星期一和星期二会用空格代替
         System.out.print("    ");

      while (date.getMonthValue() == month) {
         System.out.printf("%3d", date.getDayOfMonth());// 打印每一天
         if (date.getDayOfMonth() == today)
            System.out.print("*");// 如果是今天,就要*标记
         else
            System.out.print(" ");
         date = date.plusDays(1);
         if (date.getDayOfWeek().getValue() == 1)// 到了新的一周的星期一，换行
            System.out.println();
      }
      if (date.getDayOfWeek().getValue() != 1)
         System.out.println();
   }
}
````

# 3.8.1块作用域
块(block),由若干条java语句组成,并用一对大括号括起来。块确定了变量的作用域,块可以嵌套，但嵌套中不能声明同名变量,嵌套的块变量有相同生命周期

```` java
public class Test {
    public static void main(String[] args) {
        int n=10;
        if(n>5)
        {
            //int n=20;<- error 同名变量声明,会报错
            int k=n;
        }
        //n=k;<- error,k的作用域在if所在括号内
    }
}
````

下面的两个num不会带来编程错误,在一个作用域时确实不可以重复定义，但是一个成员变量定义后，如果在方法内定义一个局部变量，可以定义重名(**成员变量,局部变量生命周期不同,不是嵌套**)，这时候如果代码中调用到对应的变量名，默认会找最近的“变量定义”，所以这个时候可以认为成员变量被屏蔽。
```` java
class out {
    int num = 10;
    
    void show() {
        int num = 12;
        System.out.println(num);// 输出12
        System.out.println(this.num);//this调用成员变量,输出10
    }
}

public class Test {
    public static void main(String[] args) {
        out demo = new out();
        demo.show();
    }
}
````

import java.util.Scanner;

public class Test {

        int n = in.nextInt();
        if (n > 5) {
            System.out.println("n>5");// n>5时执行
        }
        else if (n == 5) {
            System.out.println("n==5");// n==5时执行
        }
        else {
            System.out.println("n<=5");// else部分可选，与最近的if组成一组，这里是else if(n==5)
        }

        int sign = 1;
        if (n <= 0) {
            if (n == 0) {
                sign = 0;
            }
            else {
                sign = -1;
            }
        }
    }
}

# 3.8.3循环
while循环，条件为真是一直执行
```` java
public class Test {
    public static void main(String[] args) {
        int num = 0;
        while (num != 0) {
            System.out.println(num);//从10打印到1
            num--;
        }
    }
}
````

do while循环，先执行语句,在循环检测条件,如果为真，继续执行
```` java
public class Test {
    public static void main(String[] args) {
        int num = 0;
        do{
            System.out.println(num);
            num--;
        }while (num != 0);/*注意while(num!=0)后面有个分号，因为
        他代表do while部分的结束，接下来是别的语句*/
    }
}
````

#3.8.4循环
for循环
第一部分对计数器初始化，第二部分检测循环条件,第三部分更新计算器
```` java
public class Test {
    public static void main(String[] args) {
        for (int i = 1; i < 10; i += 2) {
            System.out.println(i);
        }
    }
}
````
循环中检测浮点数是否相等要当心,下面是一个死循环，因为0.1不能精确表示成二进制，累加带来误差时代sum加10个0.1后却不等于1.0
```` java
public class Test {
    public static void main(String[] args) {
        double sum = 0;
        while (sum != 1.0) {
            sum += 0.1;
            System.out.println(sum);
        }
    }
}
````
# 3.8.5多重选择
switch语句将从选项值匹配的case语句执行,直到遇到break语句或者直到switch语句结束，没有匹配case，执行default语句，defalut没有也没匹配到case则switch语句直接执行结束
```` java
import java.util.Scanner;

public class Test {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        System.out.println("输入1，2，3，4之一");
        int num = in.nextInt();
        //num是选项值
        switch (num) {
        case 1:
            System.out.println("是1");
            break;
        case 2:
            System.out.println("是2");
            break;
        case 3:
            System.out.println("是3");
            break;
        case 4:
            System.out.println("是4");
            break;
        default:
            System.out.println("fuck!");
            break;
        }
    }
}
````
如果case语句结尾没有break，会执行下一个case
```` java
import java.util.Scanner;

public class Test {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        System.out.println("输入1，2，3，4之一");
        int num = in.nextInt();
        switch (num) {
        default:
            System.out.println("fuck!");
        case 1:
            System.out.println("是1");
            break;
        case 2:
            System.out.println("是2");
            break;
        case 3:
            System.out.println("是3");
            break;
        case 4:
            System.out.println("是4");
            break;

        }
    }
}
````
假如输入6,那么执行default，但是default没有break，向下执行case 1，case 1有break，执行完 case 1结束.(并不是执行完default语句就结束，只不过我们习惯把default放到最后)

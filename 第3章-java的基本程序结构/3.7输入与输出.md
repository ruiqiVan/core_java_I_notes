# 3.7.1读取输入
读取"标准输入流"System.in需要构造一个与其关联的Scanner对象  
````java
import java.util.*;/*Scanner定义在java.util中。
当使用类不在基本java.lang中,要导入*/
/**
 * This program demonstrates console input.
 * @version 1.10 2004-02-10
 * @author Cay Horstmann
 */
public class InputTest
{
   public static void main(String[] args)
   {
      Scanner in = new Scanner(System.in);//构造对象

      // get first input
      System.out.print("What is your name? ");
      String name = in.nextLine();/*nextLine读取一行输入,包括换
      行符,但返回给字符串时会把换行符丢弃*/

      // get second input
      System.out.print("How old are you? ");
      int age = in.nextInt();//读取下一个整数,不会读取换行符

      // display output on console
      System.out.println("Hello, " + name + ". Next year, you'll be " + (age + 1));
   }
}
````

如果我们调换下读取顺序
```` java
import java.util.*;
public class InputTest
{
   public static void main(String[] args)
   {
      Scanner in = new Scanner(System.in);

      
      System.out.print("How old are you? ");
      int age = in.nextInt();
        //in.nextLine();
      System.out.print("What is your name? ");
      String name = in.nextLine();

      // display output on console
      System.out.println("Hello, " + name + ". Next year, you'll be " + (age + 1));
   }
}
````
则程序会但我们输入数字按下回车后不给我们输入name,这是因为输入数字按下的回车产生的换行符nextInt不会读取,当nextLine会读取。如何修正呢?
一种解决方法是如上面所示，在两个读取语句中加一个in.nextLine();吃掉换行符

  
常用的API  
|  名称  |  作用 |
|:---:|:---:|
|**Scanner**(InputStream in)|用给定输入流创建一个Scanner对象|
|String **nextLine**()|读取一行内容,以换行符结束，会读取换行符,但返回给字符串变量中会丢弃换行符|
|String **next**()|读取下一个单词(以空格作为分隔符)|
|int **nextInt**()|读取下一个整数|
|double **nextDouble**()|读取下一个整数/浮点数|

# 3.7.2格式化输出
java5沿用了c语言函数库中的printf方法
```` java

public class Test {
    public static void main(String[] args) {
        int age = 15;
        String name = "Van";
        double money = 111.111;
        //打印为hello Van,your age is 15,your money is 111.111
        System.out.printf("hello %s,your age is %d,your money is %.3f", name, age, money);
        /*%d对应十进制整数(long int byte short均可),%s表示字符
        串,%f表示浮点数,%.3f表示小数点后打印3位*/

        //也可以使用String.format创造一个格式化字符串，方便拼接
        String All_str=String.format("hello %s,your age is %d", name, age);
        
    }
}
````

## 文件输入

```` java
import java.io.IOException;
import java.nio.charset.StandardCharsets;
import java.nio.file.Path;
import java.util.Scanner;

//当用不存在的文件构造一个Scanner,会产生输入异常,你需要在main方法标记
public class Test {
    public static void main(String[] args)throws IOException {

       //Path位于java.nio.file.Path,Path.of参数为文件路径
       //文件路径支持相对路径和绝对路径m这里为绝对路径
       //第二个参数为设置编码的方式,位于java.nio.charset.StandardCharsets

        Scanner in =new Scanner(Path.of("D:\\java_file\\myfile.txt"),StandardCharsets.UTF_8);
         
        int num=in.nextInt();
        String str=in.next();
        System.out.println(num+" "+str);
    }
}
````

## 文件输出
构造一个PrintWriter对象,需要提供文件名和编码方式,如果文件没有创建，就会创建一个，否则擦除文件内容,重写。
```` java
import java.io.IOException;
import java.io.PrintWriter;
import java.nio.charset.StandardCharsets;

public class Test {
   
    public static void main(String[] args)throws IOException {
        PrintWriter out=new PrintWriter("myfile.txt",StandardCharsets.UTF_8);
        out.print("hello world");
        out.close();
    }
}
````

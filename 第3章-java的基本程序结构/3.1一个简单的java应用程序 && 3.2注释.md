# 3.1一个简单的java应用程序
````java
public class FirstSample
{
    public static void main(String[] args) {
        \\ 注意  \"  是一个转义字符,代表一个双引号字符"
        System.out.println("We will not use \"Hello World\"");
    }
}
````

第一行的关键字**public**是一个访问修饰符,代表FirstSample对所有类可见  
关键字**class**代笔FirstSample是一个类,而java文件的代码均在类中(包括main方法,这与c++不同,c++的main函数是独立于类的)  
FirstSample是类名,java的代码区分大小写,而且java中变量名不能于java保留字相同(如 **public void**)  
并且java变量名开头不能是数字,但变量名后面可以加数字,比如0x不能充当变量名,因为0x后面加数字代表16进制,如0x11就代表十进制17  
源文件名称应该与公共类名称相同,比如这段代码所在文件名应该为FirstSample.java


---
与c++类似,java虚拟机执行代码时从main方法开始  
注意大括号{},java中用大括号划分程序的各个部分(通常称为块),块以"{"开始,"}"结束  
main方法声明格式如下  
>public class ClassName{  
    public static void main{  
        //代码  
    }  
}  

___

>System.out.println("We will not use \"Hello World\"");

java中语句以分号";"结束,通过点号"."调用方法  
**System.out.println()** 便是打印字符串到控制台中并且换行(**System.out.print()** 打印不换行) 
 




# 3.2注释
三种方法
第一种://
第二种:/* *
第三种:以/** 开始以 /*结束
比如
````java
/**
 * This is the first sample program in Core Java Chapter 3
 * @version 1.01 1997-03-22
 * @author Gary Cornell
 */
public class FirstSample
{
   public static void main(String[] args)
   {

      System.out.println("We will not use 'Hello, World!'");//这是第一种,到本行结尾
   }
   /*这样
   注释
   可以写多行*/
}
````
